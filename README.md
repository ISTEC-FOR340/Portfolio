# Edições Multimédia Interativas | Hub de Demonstração (Professor) 🚀

Bem-vindo ao repositório central com os projetos de demonstração **desenvolvidos de forma partilhada ao longo do curso de Edições Multimédia Interativas (LEM 2025/2026)** no **ISTEC**.

Esta página funciona como um laboratório vivo e um hub interativo conduzido pelo **Prof. Jorge Forero**. Os protótipos, esquemas base e simulações aqui reunidos ganharam forma em sala de aula para servirem de **inspiração e referência técnica**. A partir destas linhas de investigação práticas, cada estudante foi desafiado a explorar o código, a testar os limites da criatividade e a desenvolver os seus próprios projetos autónomos.

## 🛠️ Tecnologias e Linhas de Investigação

O ecossistema explorado cooperativamente nas aulas divide-se em quatro grandes vertentes tecnológicas:
* **Creative Coding:** p5.js (JavaScript adaptado à programação criativa e manipulação de canvas)
* **Motores 3D & Cinemática:** Unity 3D, Unity WebGL e Cinemachine
* **Interfaces de Dados:** HTML5, CSS3 Moderno (Custom Properties, CSS Grid, Flexbox) e APIs com JavaScript Assíncrono
* **WebXR:** A-Frame (WebVR) e AR.js (WebAR baseada em marcadores)

## 📂 Organização dos Projetos Modelo

O repositório reflete as bases práticas partilhadas que serviram como rampa de lançamento para o grupo de trabalho. Abaixo encontras os repositórios originais usados como ponto de partida em cada módulo:

### 🎨 1. Creative Coding & 2D Interaction
*Focado na manipulação visual e física de elementos bidimensionais.*
* **Repositório Base das Aulas:** [📦 MySketches](https://github.com/ISTEC-FOR340/MySketches)
* **Circle Matrix (Projeto 1):** Transformação de imagens numa matriz interativa de partículas que respondem dinamicamente ao rato.
* **The Weight of Words (Projeto 2):** Exploração tipográfica onde os inputs de texto ganham propriedades físicas no ecrã.

### 🎮 2. Unity 3D & Cinemachine
*Estudos de cinematografia virtual e interatividade tridimensional exportados em WebGL.*
* **Repositório Base das Aulas:** [📦 Cinemachine-WebBuild-itch](https://github.com/ISTEC-FOR340/Cinemachine-WebBuild-itch)
* **Linear Cam Example (Projeto 3):** Implementação de caminhos e calhas de câmara inteligentes com o Unity Cinemachine.
* **Enquadramentos (Projeto 4):** Estudo de realização com troca ativa de planos de câmara em tempo real aplicados a um personagem animado.

### 📊 3. Dashboards Interativos & Data
*Criação de painéis de controlo e interfaces iterativas minimalistas.*
* **Repositório Base das Aulas:** [📦 MyDashboards](https://github.com/ISTEC-FOR340/MyDashboards)
* **Weather Dashboard (Projeto 5):** Painel de controlo generativo baseado na monitorização de fluxos e variáveis digitais (*Quilled Cord Systems*).
* **AI Editor Dashboard (Projeto 6):** Visualização analítica através de uma interface web minimalista (*Candle Sushi Concept*).

### 🕶️ 4. Espaço WebXR (VR & AR)
*Experiências imersivas multiplataforma diretamente no ecossistema Web (browser).*
* **Repositório Base das Aulas:** [📦 XR](https://github.com/ISTEC-FOR340/XR)
* **VR 360 Test Explorer (Projeto 7):** Visor imersivo com navegação por hotspots e controlo por olhar (*Gaze system*) em A-Frame.
* **Bello AR Experience (Projeto 8):** Sistema de Realidade Aumentada baseado em marcadores visuais nativos (AR.js), simulando órbitas físicas tridimensionais.

## 💻 Características Técnicas do Hub

Este template coletivo foi desenhado com foco na experiência de utilizador (**UX/UI**) apresentando:
1. **Iframe Architecture:** Um visualizador integrado (`embed-container`) que permite executar qualquer sketch, dashboard ou jogo do Itch.io diretamente na página, sem quebrar a navegação principal.
2. **Design Fluido:** Interface futurista escura com gradientes dinâmicos e total responsividade para dispositivos móveis.
3. **Performance:** Código limpo e nativo, utilizando filtros CSS (`backdrop-filter`) com aceleração por hardware para o efeito de vidro fosco (*Glassmorphism*).
