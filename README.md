# Jorge Forero | Creative Coding & XR Hub 🚀

Bem-vindo ao repositório central do meu portfólio de **Projetos Académicos**, desenvolvidos no âmbito do curso de **Edições Multimédia Interativas LEM (2025/2026)** no **ISTEC**. 

Esta página funciona como um hub centralizado e interativo, permitindo explorar e executar diretamente no navegador vários esboços, simulações em 3D, dashboards de dados e experiências imersivas de Realidade Virtual (VR) e Realidade Aumentada (AR).

## 🛠️ Tecnologias e Frameworks Utilizadas

O ecossistema de projetos divide-se em quatro grandes vertentes tecnológicas:
* **Creative Coding:** p5.js (JavaScript adaptado à programação criativa)
* **Motores 3D & Cinemática:** Unity 3D, Unity WebGL e Cinemachine
* **Interfaces de Dados:** HTML5, CSS3 Moderno (Custom Properties, CSS Grid, Flexbox) e JavaScript Assíncrono
* **WebXR:** A-Frame (WebVR) e AR.js (WebAR baseada em marcadores)

## 📂 Organização dos Projetos e Sub-Repositórios

O portfólio está estruturado em 4 categorias principais. Abaixo encontras a descrição de cada projeto e as ligações para os seus respetivos repositórios de desenvolvimento:

### 🎨 1. Creative Coding & 2D Interaction
*Focado na manipulação visual e física de elementos bidimensionais.*
* **Repositório Base:** [📦 MySketches](https://github.com/ISTEC-FOR340/MySketches)
* **Circle Matrix (Projeto 1):** Transformação de imagens e dados visuais numa matriz interativa de partículas que respondem dinamicamente à posição do rato.
* **The Weight of Words (Projeto 2):** Exploração tipográfica e interativa onde os inputs de texto ganham propriedades físicas no ecrã.

### 🎮 2. Unity 3D & Cinemachine
*Estudos de cinematografia virtual e interatividade tridimensional exportados em WebGL.*
* **Repositório Base:** [📦 Cinemachine-WebBuild-itch](https://github.com/ISTEC-FOR340/Cinemachine-WebBuild-itch)
* **Linear Cam Example (Projeto 3):** Implementação de caminhos e calhas de câmara inteligentes com o Unity Cinemachine para visualizações dinâmicas.
* **Enquadramentos (Projeto 4):** Estudo cinematográfico interativo com troca ativa de planos de câmara em tempo real aplicados a um personagem animado.

### 📊 3. Interactive Dashboards & Data
*Criação de painéis de controlo e interfaces iterativas minimalistas.*
* **Repositório Base:** [📦 MyDashboards](https://github.com/ISTEC-FOR340/MyDashboards)
* **Weather Dashboard (Projeto 5):** Painel de controlo generativo baseado na monitorização de fluxos e variáveis digitais (Quilled Cord Systems).
* **AI Editor Dashboard (Projeto 6):** Exploração estética e visualização analítica através de uma interface web minimalista (Candle Sushi Concept).

### 🕶️ 4. WebXR Space (VR & AR)
*Experiências imersivas multiplataforma diretamente no ecossistema Web.*
* **Repositório Base:** [📦 XR](https://github.com/ISTEC-FOR340/XR)
* **VR 360 Test Explorer (Projeto 7):** Visor imersivo com navegação por hotspots e controlo de olhar (*Gaze system*) desenvolvido sobre a framework A-Frame.
* **Bello AR Experience (Projeto 8):** Sistema de Realidade Aumentada baseado em marcadores visuais nativos (AR.js), simulando órbitas físicas tridimensionais.

## 💻 Características Técnicas do Hub

Este portfólio foi desenhado com foco na experiência do utilizador (**UX/UI**) apresentando:
1.  **Iframe Architecture:** Um visualizador integrado (`embed-container`) que permite executar qualquer projeto (incluindo os jogos do Itch.io ou experiências em p5.js) sem nunca tirar o utilizador da página principal.
2.  **Design Fluido:** Interface futurista escura com gradientes dinâmicos no fundo e total responsividade para dispositivos móveis.
3.  **Performance:** Código limpo e nativo, utilizando *hardware acceleration* através de filtros CSS (`backdrop-filter`) para o efeito de vidro fosco (*Glassmorphism*).
